Règle de formatage du fichier CSV :

Les données doivent être écrites dans cette ordre là : titre,annee,realisateur,categorie,acteur

Chaque données est séparées d'une virgule (,)

L'année de sortie doit être écrite sous le format américain (AAAA-MM-JJ), ne pas oublier de mettre un tiret entre chaque nombre

Pour les acteurs, il faut les mettre entre guillemets et d'abord écrire le prénom puis le nom séparé par un espace.

De plus, il y a une virgule accompagnée d'un espace entre chaque acteurs différents ("nom prénom, nom prénom, nom prénom")

Voici un exemple complet de comment doit être formaté le fichier CSV :

titre,annee,realisateur,categorie,acteur

Titanic,1998-01-07,James Cameron,Romance,"Leonardo DiCaprio, Kate Winslet"
